correctMnzSat {GCDmodel}R Documentation

Correction for monazite saturation, Montel 1992 w. elements partitionning

Description

Correction for monazite saturation, Montel 1992 w. elements partitionning

Usage

correctMnzSat(kd, c0, pm, cmins, min.props, melt.arg = list(),
  dont = character(0))

Arguments

kd

Named matrix. Partition coefficients. Must include Monazite (Mnz)

pm

Scalar. Degree of partial melting (0-100)

cmins

Matrix. Composition of minerals (not used, for consistency)

min.props

Named vector. Mineral proportions (normalized to 1)

melt.arg

List. Further useful arguments. Includes here the melt major elements (melt.arg$mjrs),trace (melt.arg$trc), water (melt.arg$H2O), temperature in KELVIN (melt.arg$TT)

dont

Character vector. Do not calculate for these elements. Use to avoid overptinting previous saturation calc (i.e. if you use a saturation routine /after/ this one, )
